Техническое задание
Проект: Управление задачами

Описание:
Разработать веб-приложение для управления задачами. Пользователи могут создавать, просматривать, редактировать и удалять задачи.

Технологии:

Язык программирования: Go
Фреймворк: Fiber
ORM: GORM
База данных: PostgreSQL
Контейнеризация: Docker
Функциональность:

Создание задачи

Метод: POST
Путь: /tasks
Параметры: Заголовок, Описание, Статус (новая, в работе, завершена)
Получение списка задач

Метод: GET
Путь: /tasks
Параметры: Нет
Получение задачи по идентификатору

Метод: GET
Путь: /tasks/{id}
Параметры: id (идентификатор задачи)
Редактирование задачи

Метод: PUT
Путь: /tasks/{id}
Параметры: id (идентификатор задачи), Заголовок, Описание, Статус
Удаление задачи

Метод: DELETE
Путь: /tasks/{id}
Параметры: id (идентификатор задачи)
Получение списка задач по статусу

Метод: GET
Путь: /tasks/status/{status}
Параметры: status (статус задачи)
Создание пользователя

Метод: POST
Путь: /users
Параметры: Имя, Фамилия, Электронная почта
Получение списка пользователей

Метод: GET
Путь: /users
Параметры: Нет
Получение пользователя по идентификатору

Метод: GET
Путь: /users/{id}
Параметры: id (идентификатор пользователя)
Редактирование пользователя

Метод: PUT
Путь: /users/{id}
Параметры: id (идентификатор пользователя), Имя, Фамилия, Электронная почта
Удаление пользователя

Метод: DELETE
Путь: /users/{id}
Параметры: id (идентификатор пользователя)
Связь задач с пользователями

Реализовать связь многие ко многим между задачами и пользователями через таблицу user_tasks
Рефакторинг на Fiber
Замените существующий код с использованием Gin на Fiber.
Обновите маршруты, обработчики и структуры данных в соответствии с Fiber.
ORM-ассоциации и хранение в SQL
Создайте модели Task и User с использованием GORM.
Установите связь многие ко многим между Task и User через таблицу user_tasks.
Docker
Создайте Dockerfile для приложения.
Создайте docker-compose.yml для развертывания контейнера приложения и контейнера базы данных PostgreSQL.
Это базовый пример, и вам, вероятно, потребуется настроить дополнительные параметры в зависимости от особенностей вашего проекта.
